# PetPlanet

PetPlanet 是一款高级温馨的宠物生活管理 App，集宠物档案、日常照护、健康记录、体重消耗与 2D 成长回忆于一体。

## 当前产品结构

主导航保持五个清晰入口：

- **首页**：当前宠物总览、最近 3 条关键提醒、快捷记录、今日照护进度、月度 Bento 和回忆预览。
- **日常**：集中管理待办与提醒，并记录喂食、饮水、体重、生活照片和月度消耗。
- **健康**：管理病历、复诊、用药、照片与筛选搜索。
- **回忆**：展示 2D 成长相册、闪光回忆和季节时间线；真实 3D 仅保留为未来方向，目前不加载 3D 场景。
- **档案**：维护宠物资料、头像，并支持添加、编辑和删除宠物。

当前宠物切换后，全站数据同步；全局保留一个快速记录入口。首页只展示提醒预览，新增、完成、稍后和删除统一在日常页处理。

## 技术与数据边界

前端使用 React、TypeScript 和 Vite，界面采用奶油白、柔雾橘、灰紫与深墨紫组成的 Bento / 2.5D 卡片设计系统。

默认数据保存在浏览器 `localStorage`，无需后端即可完整体验。仓库同时保留可选的 Express + SQLite 基础设施，后续 API 工作边界见 [后端交接说明](docs/backend-handoff.md)。

## 本地运行

```bash
npm install
npm run dev
```

访问 `http://127.0.0.1:61413/`。可通过 `.env.example` 了解可选 API 模式的环境变量约定。
